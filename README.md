# Калькулятор на C++ (для macOS)
Графический калькулятор, разработанный в рамках учебной практики на "сложном" уровне с неким творческим элементом 🙈. 

## Технологии
- C++
- Cocoa
- Objective-C++

## Возможности
- Ввод арифметических выражений
- Сложение, вычитание, умножение и деление
- Поддержка скобок
- Учёт приоритетов операторов
- Возведение в степень
- Вычисление корня
- Обработка ошибок ввода

## Архитектура
Интерфейс приложения реализован при помощи Cocoa, вычислительная логика написана на C++, взаимодействие между ними осуществляется через Objective-C++.
