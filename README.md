# Калькулятор на C++ (для macOS)
Графический калькулятор, разработанный в рамках учебной практики на "сложном" уровне с неким творческим элементом 🙈. 

## Технологии
- C++
- Cocoa
- Objective-C++

## Возможности
- Ввод арифметических выражений
- Сложение, вычитание, умножение и деление
- Поддержка скобок
- Учёт приоритетов операторов
- Возведение в степень
- Вычисление корня
- Обработка ошибок ввода

## Архитектура
Интерфейс приложения реализован при помощи Cocoa, вычислительная логика написана на C++, взаимодействие между ними осуществляется через Objective-C++.

## Как запустить
Нужен Mac — программа использует Cocoa, так что под Windows/Linux она не соберётся.

Самый быстрый способ — через терминал. Заходим в папку с проектом и запускаем скрипт сборки:

```sh
cd путь/до/папки/Calculator67
./build.sh
```

Скрипт сам всё скомпилирует (через `clang++`, он идёт вместе с Xcode) и сразу откроет окно калькулятора. Если терминал ругается `permission denied`, дайте скрипту право на запуск один раз:

```sh
chmod +x build.sh
```

Можно и без скрипта, одной командой:

```sh
clang++ -std=c++17 -fobjc-arc -framework Cocoa main.mm -o Calculator && ./Calculator
```

Кто хочет открыть в Xcode: создаём проект **Command Line Tool** на C++, кидаем туда `main.mm`, в **Build Phases → Link Binary With Libraries** добавляем `Cocoa.framework` и жмём ▶︎.

> Если запускаете не через `build.sh`, держите рядом папку `assets` — из неё подгружаются гифки. Иначе всё посчитает, просто без них.

## Как пользоваться
Набираете выражение кнопками и жмёте `=`. Работают скобки, степень `^` и корень `√`, можно писать дробные через запятую или точку. А ещё мы спрятали пару пасхалок — попробуйте получить в ответе 42, 52 или 67 :)
